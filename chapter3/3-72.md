```
--------- s1
|   e1   |
---------- p + n*8
|        |
|        |
|        |
---------- p
|   e2   |
---------- s2
```

A.
当n为奇数时，s2 = s1 - (24 + 8*n)
当n为偶数时，s2 = s1 - (16 + 8*n)
道理就是要向下取整为16的倍数

B.
p = (s2 + 15) & 0xfffffff0，向下取整为16的倍数

C.
可以通过给s2赋值来理解：
s2=0,p=0, e2=p-s2=0
s2=1,p=16,e2=p-s2=15
s2=2,p=16,e2=p-s2=14
...
s2=15,p=16,e2=p-s2=1
s2=16,p=16,e2=p-s2=0

要使e1的值最小，则需使e1+e2最小同时e2最大。当n为偶数时，e1+e2=16最小；当s1%16=1时，e2=15最大。

即：n为偶数，s1%16=1时e1最小。

要使e1的值最大，则需使e1+e2最大同时e2最小。当n为奇数时，e1+e2=24最大；当s1%16=0时，e2=0最小。

即：n为奇数，s1%16=0时e1最大。

D.
s2对齐为16的倍数，p对齐为16的倍数
