根据汇编代码：

str1的栈结构
```
--------------
|     y      |
-------------- 184
|  x[A][B]   |
-------------- 176
|  x[A][B]   |
|  x[A][B]   |
-------------- 0
```

str2的栈结构
```
--------------
|     u      |
-------------- 32
|    s[A]    |
-------------- 24
|    s[A]    |
-------------- 16
|  t  | s[A] |
-------------- 8
|  array[B]  |
-------------- 0
```


由于64位机器，栈数据是8字节对齐，所以可以得出一个范围值：<br>
4bytes < array[B] <= 8bytes<br>
12bytes < s[A] <= 20bytes<br>
176bytes < x[A][B] <= 184bytes<br>
所以<br>
6 < a <= 10<br>
4 < b <= 8<br>
44 < a*b <= 46<br>
所以<br>
a=6,b=9<br>
