for(i = 0; i < 16; i++) {
    for(j = 0; j < 16; j++) {
        square[i][j].y = 1;
    }
}
for(i = 0; i < 16; i++) {
    for(j = 0; j < 16; j++) {
        square[i][j].c = 0;

    }
}

因为每个块能缓存两个数组元素，每个元素四个值，所以我们分开来看：
第一部分写256次，有一半未命中，未命中128次
