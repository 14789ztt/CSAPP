cache大小2048B，直接映射每组1行，块大小为32B，块偏移为5位，缓存组数64，组索引为6位
```c
for(i = 0; i < 16; i++) {
    for(j = 0; j < 16; j++) {
        square[j][i].c = 0;
        square[j][i].m = 0;
        square[j][i].y = 1;
        square[j][i].k = 0;
    }
}
```
A.写总数是1024 <br/>
B.因为只能保存数组的1/2，所以第一部分加载到cache后，又会被第二部分替换，所以每一列不同行的元素都要重新加载到缓存 <br/>
但是数组元素内部c会不命中，m y k会命中，所以会有1/4次未命中，未命中次数256 <br/>
C.不命中率25% <br/>
