```c
/* $begin forkprob7 */
#include "csapp.h"
int counter = 1;

int main() 
{
    if (fork() == 0) {
	counter--;  
	exit(0);
    }
    else {
	Wait(NULL); 
	printf("counter = %d\n", ++counter);
    }
    exit(0);
}
/* $end forkprob7 */

```
进程图：
```
main---->fork---->wait---------------->printf:++counter---->exit
          |                     |
          ---->counter-- ---->exit
```
因为子进程得到与父进程用户级虚拟地址空间相同的（但是独立的）一份副本，包括代码和数据段、堆、共享库以及用户栈<br/>
所以父进程的counter不受子进程影响，输出2
